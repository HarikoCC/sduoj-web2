# SDUOJ 新版前端

之前的前端，管理端与用户端是分离的，使用了 Vue 作为开发的框架，
由于使用的 UI 框架有一些 Bug 长时间无人维护，所以决定使用 React 与 Ant Design
重新进行编写。

> 项目正在施工中，预计完成时间为 2022 年 7 月
